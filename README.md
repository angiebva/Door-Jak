# Door-Jack
First mini web-game to practice/learn using html css and javascript. <br><br>


Rules: <br>
Wojaks keep showing up to your house. <br>
Invite the friendly wojaks into your home. <br>
Kick the strange wojaks out! <br> <br>

Lose all hearts = Game Over!💔 <br>
Get 10 correct = You Win 🎉 <br>
